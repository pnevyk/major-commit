# Major-commit

[Majordomo](https://github.com/nevyk/majordomo) command for git commiting in a more comfortable way.

## Installation

```bash
$ npm install major-commit
```

## Usage

```bash
$ majordomo commit
```

## Modules

### git

- adds specified files to index
- removes specified from index
- commits changes
